[中华人民共和国民政部-行政区划代码](https://www.mca.gov.cn/n156/n186/index.html)
[中华人民共和国国家统计局-统计用区划代码和城乡划分代码编制规则](https://www.stats.gov.cn/sj/tjbz/gjtjbz/202302/t20230213_1902741.html)

[国家标准全文公开系统 GB 11643-1999](https://openstd.samr.gov.cn/bzgk/gb/newGbInfo?hcno=080D6FBF2BB468F9007657F26D60013E)

本工具提供中国大陆第二代身份证号码生成功能，可以根据指定的条件，如：出身地、出身日期、性别等生成符合中国大陆 GB 11643-1999 国标的身份证号码。本工具生成的结果不代表真实的身份证号码，仅供参考（请勿用于非法用途）。
